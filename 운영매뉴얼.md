# 로뎀카페 · 선지서점 홈페이지 운영 매뉴얼

## 사이트 구성
- **index.html** — 홈 / **cafe.html** — 카페 / **book.html** — 서점
- **data.json** — 소식·시즌메뉴·신간 내용이 담긴 데이터 파일
- **admin.html** — 관리자 페이지 (아래 내용을 폼으로 수정)
- **img/season1.jpg, img/season2.jpg** — 시즌메뉴 사진

## ⭐ 매월 할 일 — 관리자 페이지에서 (코드 몰라도 OK)
1. `https://ezerahn.github.io/gsmch-mission/admin.html` 접속
2. 이달의 소식 / 시즌 추천 메뉴(사진 포함) / 신간 / 담임목사님 저서 수정
3. 맨 아래 **[저장하고 사이트에 반영]** 클릭 → 1~2분 뒤 반영

## 🔑 GitHub 토큰(열쇠) 발급 — 처음 한 번만
1. github.com 로그인 → 우측 상단 프로필 사진 → **Settings**
2. 왼쪽 맨 아래 **Developer settings** → **Personal access tokens** → **Fine-grained tokens** → **Generate new token**
3. 설정:
   - Token name: `gsmch-admin` (아무 이름)
   - Expiration: 1년 (만료되면 재발급)
   - Repository access: **Only select repositories** → `gsmch-mission` 선택
   - Permissions → Repository permissions → **Contents: Read and write** (이것만!)
4. **Generate token** → 나오는 긴 문자열 복사
5. admin.html 상단 토큰 칸에 붙여넣기 (그 기기에 자동 저장됨)

⚠️ 토큰은 비밀번호처럼 취급하세요. 담당자가 바뀌면 GitHub에서 기존 토큰 삭제 후 새로 발급.

## 메뉴판 가격 수정 (관리자 페이지 대상 아님)
cafe.html을 GitHub에서 직접 수정: 파일 열기 → 연필 아이콘 → 가격 숫자만 수정 → Commit changes.
메뉴 추가는 한 줄 복사 후 이름·가격만 변경:
```html
<div class="mi"><span class="n">메뉴이름</span><span class="p">3,000</span></div>
```

## 담당자 인수인계
- 새 담당자 GitHub 계정 → 저장소 Settings → Collaborators 초대 (또는 Transfer로 이전)
- 새 담당자 명의로 토큰 재발급 → admin.html에 등록
- 이 매뉴얼 전달

## 주의사항
- 반영이 안 보이면: 모바일은 당겨서 새로고침, PC는 Cmd+Shift+R
- HTML 직접 수정 시 `< >` 태그는 건드리지 말고 글자·숫자만
- 가격·운영시간 변경 시 사이트도 꼭 함께 업데이트
